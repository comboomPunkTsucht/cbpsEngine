## Features at a Glance

✅ **Entity Component System** – Flexible, modular game objects
✅ **3D-First** – Uses JOML matrices, 2D via orthographic camera
✅ **Input Handling** – Thread-safe keyboard & mouse events
✅ **Frame Timing** – Configurable FPS with delta-time clamping
✅ **Cross-Platform** – Windows, macOS, Linux
